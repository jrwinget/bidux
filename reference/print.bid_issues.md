# Print method for bid_issues objects

Displays a triage view of telemetry issues with severity-based
prioritization and provides a reminder about legacy list access for
backward compatibility.

## Usage

``` r
# S3 method for class 'bid_issues'
print(x, ...)
```

## Arguments

- x:

  A bid_issues object from bid_ingest_telemetry()

- ...:

  Additional arguments (unused)

## Value

Invisible x (for chaining)
