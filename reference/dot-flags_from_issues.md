# Extract global telemetry flags from issues and events

Extract global telemetry flags from issues and events

## Usage

``` r
.flags_from_issues(issues_tbl, events, thresholds)
```

## Arguments

- issues_tbl:

  Tidy issues tibble

- events:

  Raw events data frame

- thresholds:

  Threshold parameters used in analysis

## Value

Named list of boolean flags
